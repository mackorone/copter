# copter

A simple copter game built with [`pynput`](https://pynput.readthedocs.io)

## Quick start

```
$ python3.8 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python copter.py
```

Press any button to fly higher. Press `CTRL-C` to exit.
